# TokenFlow — AI API Cost Intelligence & Optimization

TokenFlow is a production-grade SaaS platform designed to help companies monitor, analyze, and optimize their AI/LLM API expenditures across multiple providers (OpenAI, Anthropic, Google, etc.).

## 🚀 Key Features

- **Unified Dashboard**: Real-time visibility into costs across all AI providers.
- **Smart Routing**: Automatically route requests to the most cost-effective provider without sacrificing quality.
- **Cost Analytics**: Detailed breakdowns by model, team, and endpoint.
- **AI Optimization**: Automated suggestions to compress prompts and choose better model tiers.
- **Budget Guards**: Set hard limits and receive intelligent alerts before budgets are exceeded.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS with a custom-built glassmorphism design system.
- **Visualizations**: [Chart.js](https://www.chartjs.org/) for high-performance data visualization.
- **Icons**: [Lucide React](https://lucide.dev/) for a clean, modern interface.

## 📦 Getting Started

1. **Clone and Install**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Open Dashboard**:
   Navigate to [http://localhost:3000/dashboard](http://localhost:3000/dashboard) to see the platform in action.
